## 1.0.0

* Initial release.
* 3D tilted elliptical ring chart with perspective depth effect.
* Fully customizable: stroke width, gap, tilt, colors, shadow, glow, indicator line.
* Built-in interactive legend with top / bottom / left / right / none positioning.
* Smooth animated transitions between selected segments.
* Automatic light/dark theme adaptation with per-segment color overrides.
* Zero-value segment handling (show or hide).
* Custom legend value formatter.
* Tap callback with segment index and data.
* Supports 1 to 1000+ segments.
* Comprehensive documentation and 12 example demos.
